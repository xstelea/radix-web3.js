# Radix Web3.js

JavaScript and TypeScript packages for building Radix applications, wallet
connections, Gateway integrations, transaction workflows, and agent-friendly
tooling.

## Packages

- [`radix-web3.js`](./packages/core) for promise-based Radix client utilities.
- [`radix-connect`](./packages/connect) for Radix Wallet interaction requests.
- [`@radix-effects/gateway`](./packages/gateway) for Effect-based Gateway reads.
- [`@radix-effects/tx-tool`](./packages/tx-tool) for Effect transaction workflows.
- [`@radix-effects/transaction-stream`](./packages/transaction-stream) for polling Gateway transaction streams.
- [`rdx-cli`](./packages/cli) for agent-first manifest workflows.
- [`radix-agent-toolkit`](./packages/agent-toolkit) for GOAT SDK Radix tools.
- [`sbor-ez-mode`](./packages/sbor-ez-mode) for parsing programmatic SBOR values.
- [`@radix-effects/shared`](./packages/shared) for shared branded types and schemas.

## Quick Start

Install the core client:

```shell
npm install radix-web3.js
```

## Example Usage

Create a keypair, derive an account address, submit a manifest, and read
balances:

```typescript
import {
  createEd25519KeyPair,
  createRadixWeb3Client,
  deriveAccountAddressFromPublicKey,
  manifests,
} from 'radix-web3.js';

const keyPair = createEd25519KeyPair();

const client = createRadixWeb3Client({
  networkId: 'Stokenet',
  notaryPublicKey: keyPair.publicKey(),
  notarizer: (hash) => keyPair.signToSignature(hash),
});

const accountAddress = await deriveAccountAddressFromPublicKey(
  keyPair.publicKey(),
  2,
);

const { transactionId } = await client.submitTransaction(
  manifests.sendResourceManifest({
    fromAddress: accountAddress,
    toAddress: 'account_tdx_2_12yf3qp4feqnw...',
    resourceAddress:
      'resource_tdx_2_1tknxxxxxxxxxradxrdxxxxxxxxx009923554798xxxxxxxxxtfd2jc',
    amount: '1',
  }),
);

const { fungibleTokens } = await client.getBalances(accountAddress);
console.log(transactionId, fungibleTokens);
```
